# Aula 2: GitHub Actions 🚀
## Automatizando o processo
---
### Objetivos
- Configurar o repositório
- Entender CI (Continuous Integration)
- Criar o arquivo `.github/workflows/main.yml`
---
### Conceitos Chave
- **Events**: O gatilho (push na main)
- **Jobs**: O que deve ser feito (Build)
- **Steps**: Os passos individuais
